# UCAS Classer 1.0.2

本次更新主要聚焦稳定性与可用性收口。

## 更新内容
- 修复部分课程作业未正确识别与导入的问题
- 补齐“待做”作业的详情入口
- 通知与作业原始页面统一改为带登录态打开
- 优化自动 `collect` 调度，启动后会主动执行一轮 `check + collect`
- 支持分别设置 `Check / Collect / Cookie` 刷新间隔
- 设置页重排：
  - 下载目录单独一行
  - 课程范围点击即生效
  - 三个刷新时间更紧凑
- 托盘与关闭常驻行为继续收口，适合长期后台使用

## 说明
- 登录仍保留浏览器参与，但登录后的检查、课程刷新、内容采集与下载已经基本走 API / request 路线。
- 本版本仍以 Windows 桌面端为主。
