## 2026-03-13 / v1.0.4 当前状态

- 自动侧收 MVP 已落地：当前只支持左右边缘，默认关闭，可在设置中开启。
- 贴边判定已放宽为“贴到边缘或越过边缘也算命中”，避免无边框窗口外框和视觉边缘不一致导致漏触发。
- 托盘重新打开窗口时会恢复正常展开尺寸，不再保留异常窄条尺寸。
- 侧收收起/展开已加入窗口插值动画与前端淡出过渡，当前进入手感打磨阶段。
- `v1.0.4` 发版基线已完成，当前文档状态与打包端行为对齐。

# v1.0.1 -> v1.1.0 进度

## 2026-03-12 / v1.0.3 当前状态
- 下载目录已支持系统文件夹选择器。
- 已支持课程分目录，配置写入 `app-settings.json`，下载时按“课程子目录 + 资料树父级目录”生效。
- 已支持资料批量下载，资料树中的嵌套文件夹会在本地自动创建并保留层级。
- 批量下载状态已接入主界面状态区：`Waiting / Downloading / Success / Fail`，其中 `Success` 20 秒后自动回到 `Waiting`。
- 开发端与打包端已统一到单向同步机制；打包端仍保留系统路径存储约束。
- 自动 collect 已拆成 `full / summary` 两种模式：启动与手动 collect 走 `full`，后台常态走 `summary`，summary 发现 diff 后下一次自动 collect 自动切回 `full`。

## 本轮已推进
- 托盘化与关闭后常驻
- `check / collect / cookie refresh` 时间可自定义
- `last check / last collect` 已持久化，重开不再丢失
- 设置弹窗重排：
  - 下载目录单独一行
  - 课程范围点击即生效
  - 自动侧收开关点击即生效
  - 三个时间设置压成一行
- 自动 `collect` 调度重新梳理，启动时会主动做一轮 `check + collect`
- 通知与作业原始页面打开统一改为注入登录态后用 Edge 打开
- 作业详情补齐了“待做”类型入口
- 下载目录支持系统文件夹选择器
- 支持课程分目录，下载时按“课程子目录 + 资料树父级目录”落盘
- 支持资料批量下载，并保留资料树中的嵌套文件夹结构
- 批量下载状态已接入主界面状态区
- 自动采集已改为“粗采探测 + 下一次全量回补”模式，降低后台常态网络负载
