# v1.0.1 -> v1.1.0 进度

## 本轮已推进
- 托盘化与关闭后常驻
- `check / collect / cookie refresh` 时间可自定义
- `last check / last collect` 已持久化，重开不再丢失
- 设置弹窗重排：
  - 下载目录单独一行
  - 课程范围点击即生效
  - 三个时间设置压成一行
- 自动 `collect` 调度重新梳理，启动时会主动做一轮 `check + collect`
- 通知与作业原始页面打开统一改为注入登录态后用 Edge 打开
- 作业详情补齐了“待做”类型入口

## 已确认可继续推进的方向
- 下载目录支持资源管理器选文件夹
- 下载目录分目录设计
- 批量下载资料
- 资料自动同步与展示细节继续完善

## 暂时后放
- 通知与作业详情完全内嵌展示
- 作业代提交
- SEP 其他能力扩展
- 长时间不活跃后的自动侧收
- 到时提醒、新通知自动提醒
- 动画与纯视觉优化

## 当前版本目标理解
`1.1.0` 更适合定义为：

- 更适合长期挂着用
- 下载体验更顺手
- 设置项更完整
- 托盘与后台调度更稳定
