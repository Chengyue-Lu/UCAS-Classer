## UCAS Classer 使用说明

### 一、首次使用
1. 首次需要登录时，程序会自动打开浏览器进入 SEP 登录页。
2. 登录完成后，停留在已登录页面即可，程序会自动保存登录态。
3. 应用启动后会自动执行一次 Check，并在登录态有效时执行一次完整 Collect。

### 二、主界面按钮
1. Check：检查当前登录态是否仍然有效。
2. Collect：立即执行一次完整采集，并自动导入本地数据库。
3. Login：重新进入登录流程，适用于登录态失效或需要切换账号。

### 三、设置说明
1. 下载目录支持直接手动填写，也支持系统文件夹选择器。
2. 支持课程分目录：可为每门课单独指定主下载目录下的子文件夹。
3. 课程范围支持切换为：全部、当前学期、以前学期。
4. Check / Collect / Cookie 刷新时间支持自定义。
5. 自动侧收默认关闭；开启后，窗口拖到屏幕左侧或右侧边缘会自动收起为边缘短条。

### 四、下载说明
1. 单个资料或附件可直接下载到本地。
2. 资料模块支持批量下载。
3. 批量下载会保留资料树中的子文件夹层级。
4. 若设置了课程分目录，实际下载位置为：
    主下载目录 / 课程子目录 / 资料原有子路径 / 文件

### 五、后台与托盘
1. 关闭窗口后，程序默认继续驻留在系统托盘，不会直接退出。
2. 可通过托盘菜单重新打开主窗口或退出程序。

### 六、存储位置
1. 登录态、设置、数据库：
    `%APPDATA%\UCAS Classer`
2. 采集缓存：
    `%LOCALAPPDATA%\UCAS Classer\cache`

### 七、如果出现异常
1. 先点击 Check，确认登录态是否有效。
2. 如 Check 失败，再点击 Login 重新登录。
3. 如 Collect 后仍未刷新，等待当前任务完成后再手动执行一次 Collect。
4. 如下载失败，先检查下载目录是否可写，再确认登录态是否仍然有效。
